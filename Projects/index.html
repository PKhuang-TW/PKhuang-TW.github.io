APB UVM VIP

[Github]
A reusable UVM-based verification IP for the AMBA APB protocol [IHI0024E]. It provides configurable master and slave BFMs with SVA protocol checks, functional coverage, and a scoreboard for data and response validation.

Sync FIFO UVM VIP

[Github]
A reusable UVM-based verification IP for synchronous FIFO. It includes configurable depth/width, active and passive agents, scoreboard with reference model, and functional coverage. Supports blocking read/write and checks full/empty behavior and data integrity.

Counter UVM VIP

[Github]
A simple 3-bit up-down counter with direction control and edge-triggered reverse logic. It counts from 0 to 7 and back, automatically changing direction at the boundaries. Supports manual direction reversal through a one-cycle reverse signal.
